# 外部開發工作空間技能清單 (External Skills Inventory)

此目錄存放自外部開源社群或第三方引入的技能包與設計模板。

## 外部技能列表

| 技能名稱 | 來源 / 連結 | 說明 | MCP |
| :--- | :--- | :--- | :--- |
| `Awesome-PPT-Design-Skills-main` | [GitHub 來源](https://github.com/software-ai-life/Awesome-PPT-Design-Skills/tree/main) | PPT 簡報設計與排版視覺優化技能包 | No MCP |

## 內部開發技能
- 本地開發的專屬功能性技能（例如瀏覽器控制、MCP 伺服器建置等）存放在內部技能目錄中。
- 請參閱：[內部開發技能目錄](../skills_internal/README.md)

## 管理規則
- **新增/刪減**：僅在引進新的外部技能或版本更新時，才需更新此文件。
- **讀取限制**：根據 `.init_agent`，Agent 平時不會主動讀取此目錄，僅在需要使用特定技能或進行清單維護時才開啟。
