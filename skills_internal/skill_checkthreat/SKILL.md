# 🛡️ Skill: Threat Check (安全性威脅檢查)

此技能定義了工作空間中所有「新加入技能 (Skills)」必須通過的安全審計標準，以確保環境的安全與純淨。

## 🎯 核心功能
1. **靜態安全掃描**：檢查技能資料夾中是否含有可執行副檔名（`.exe`, `.sh`, `.py`, `.js`, `.bat`, `.vbs`, `.ps1`）。
2. **行為審計**：掃描檔案內容是否包含網路傳輸邏輯（如 `http`, `curl`, `fetch`, `requests`, `upload`, `download`）。
3. **主動防禦**：嚴格禁止技能具備自行下載或上傳任何檔案的能力。

## 🛠️ 使用規範
- **執行時機**：任何新的技能包（如從 GitHub 下載）進入 `skills/` 目錄前或後，必須優先執行此檢查。
- **檢查結果**：檢查完成後應提供「安全報告」，確認無威脅後方可正式登錄至技能清單。
- **負面列表 (Forbidden)**：
    - 嚴禁出現自動聯網代碼。
    - 嚴禁出現二進位可執行文件（除非經特別核准）。

## 📜 審計流程示例
- 使用 `grep_search` 掃描敏感關鍵字。
- 使用 `list_dir` 遍歷所有層級之副檔名。
